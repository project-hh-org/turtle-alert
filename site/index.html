🎯 이런 앱이에요

하루 종일 모니터 들여다보고 있으면 어느새 거북이 자세가 되죠. 거북이경보는 메뉴바에 조용히 상주하면서 정한 시간마다 스트레칭하라고 잔소리합니다.

🕐 실시간 남은 시간 메뉴바에 🐢 29:59 형태로 다음 알림까지 남은 시간 표시
🚨 네이티브 알림 macOS 알림센터에 도달 + 랜덤 스트레칭 가이드 제공
⚙️ 간격 조절 15분 / 30분 / 45분 / 1시간 중 선택
📊 일일 카운트 오늘 몇 번이나 스트레칭했는지 자동 집계
💾 설정 저장 앱 재시작해도 간격 유지
🪶 가벼움 창 없이 메뉴바 아이콘만. Dock도 차지하지 않음

📦 설치하기

✅ 방법 1. Homebrew (권장)

터미널에 익숙하다면 한 줄이면 끝. 칩셋 자동 인식 + Gatekeeper 우회까지 전부 처리됩니다.

brew install --cask project-hh-org/turtle-alert/turtle-alert

제거: brew uninstall --cask turtle-alert

📥 방법 2. DMG 직접 다운로드

위에서 내 Mac 칩에 맞는 DMG를 받으세요. 터미널에서 uname -m 실행하면 칩을 알 수 있어요 (arm64 = Apple Silicon, x86_64 = Intel).

DMG를 더블클릭 → 거북이경보 아이콘을 Applications 폴더로 드래그하세요.

⚠️ 중요 — Apple Developer ID로 서명되지 않아 Gatekeeper가 "손상되어 열 수 없습니다" 경고를 표시합니다. 아래 한 줄로 격리 속성을 제거하세요. 
xattr -dr com.apple.quarantine /Applications/TurtleAlert.app

이후부터는 Finder 더블클릭으로 정상 실행됩니다.

🚀 사용 방법

  1. 앱을 실행하면 상단 메뉴바에 🐢 이모지가 등장합니다.
  2. 🐢 아이콘을 클릭하면 메뉴가 나타납니다.
  3. "시작"을 누르면 타이머가 작동합니다.
  4. "알림 간격" 서브메뉴에서 원하는 주기를 선택합니다.
  5. 설정한 시간마다 🚨 거북이경보 발령! 알림과 함께 랜덤 스트레칭 가이드가 도착합니다.
